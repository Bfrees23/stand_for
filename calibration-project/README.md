# Система Калибровки Корректоров

Полноценное веб-приложение для автоматизации калибровки корректоров с поддержкой работы через последовательные порты (COM/USB).

## 📋 Описание

Система предназначена для проведения полной калибровки корректоров давления с автоматическим контролем всех этапов:

1. **Калибровка температуры** (-40°C, +10°C, +60°C) с использованием МИТ-8
2. **Проверка импульсов** (20 импульсов)
3. **Проверка датчиков давления** (абсолютный/перепад)
4. **Генерация отчета** с возможностью экспорта в PDF

## 🔧 Конфигурация

### Переменные окружения

Для настройки подключения к БД используйте переменные окружения в `docker-compose.yml`:

- `DB_HOST` - хост БД (по умолчанию: db)
- `DB_PORT` - порт БД (по умолчанию: 3306)
- `DB_NAME` - имя БД (по умолчанию: calibration_db)
- `DB_USER` - пользователь БД (по умолчанию: calibration_user)
- `DB_PASS` - пароль БД (по умолчанию: calibration_password)

### Подключение оборудования

Для работы с реальными COM-портами необходимо:

1. Пробросить устройства в контейнер:
```yaml
devices:
  - /dev/ttyUSB0:/dev/ttyUSB0
  - /dev/ttyUSB1:/dev/ttyUSB1
```

2. Добавить права пользователю www-data:
```bash
usermod -a -G dialout www-data
```

## 📊 База данных

Проект использует MySQL 8.0 со следующей структурой:

- **users** - пользователи системы
- **devices_config** - конфигурация устройств
- **calibration_sessions** - сессии калибровки
- **temperature_points** - точки калибровки температуры
- **temperature_verification** - результаты поверки температуры
- **impulse_checks** - проверка импульсов
- **pressure_points** - точки проверки давления
- **session_logs** - журнал действий
- **reports** - отчеты

## 🎯 Основные возможности

### 1. Настройки стенда
- Выбор COM-портов для всех устройств
- Настройка скорости обмена (9600-115200 бод)
- Проверка связи с каждым устройством
- Настройка диапазонов давления

### 2. Калибровка температуры
- Пошаговый мастер для 3 точек (-40, +10, +60°C)
- Автоматический опрос МИТ-8 каждые 2 секунды
- Таймер стабилизации (5 минут)
- Визуализация графика температуры
- Запись точек в корректор

### 3. Проверка импульсов
- Фиксация 20 импульсов
- Прогресс-бар выполнения
- Список зафиксированных импульсов

### 4. Проверка давления
- Автоматический расчет точек (5 для абсолютного, 3 для перепада)
- Контроль попыток (максимум 3)
- Автоматическое определение брака
- Визуальная индикация статуса точек

### 5. Отчетность
- Полный протокол калибровки
- Таблицы с результатами по всем этапам
- Экспорт в PDF (требуется установка TCPDF/mPDF)
- Сохранение в базу данных
- Журнал всех действий с временными метками

## 🔒 Безопасность

- Таймаут бездействия (15 минут) с предупреждением
- Полное логирование всех действий
- Валидация входных данных
- Защита от SQL-инъекций (PDO prepared statements)
- Управление сессиями

## 🛠️ Расширение функциональности

### Добавление поддержки реальных устройств

Отредактируйте `src/includes/SerialPortHandler.php`:

```php
public function readMIT8Temperature($channel = 1) {
    // Реальная команда для МИТ-8
    $command = sprintf("READ_CH%d\n", $channel);
    $this->write($command);
    $response = $this->read(1000);
    return $this->parseMIT8Response($response);
}
```

### Интеграция с PDF-библиотекой

Установите TCPDF:
```bash
composer require tecnickcom/tcpdf
```

Или mPDF:
```bash
composer require mpdf/mpdf
```
