# Система калибровки корректоров

Веб-приложение для автоматизированной калибровки и поверки датчиков температуры и давления.

## Общая архитектура

Система выступает в роли единого пульта управления и взаимодействует с тремя типами устройств через последовательные порты (USB/COM):

1. **Калибратор температуры (МИТ-8)** - считывает эталонные значения температуры
2. **Корректор (тестируемое устройство)** - отправка команд на запись точек калибровки, считывание внутренних показаний датчиков
3. **Калибратор давления** - опрос оператора о выставленных значениях давления или автоматическое считывание

## Требования

- PHP 7.4 или выше
- Веб-сервер (Apache/Nginx)
- Доступ к последовательным портам (COM/USB)
- Расширения PHP: php_serial (для работы с COM-портами)

## Установка

1. Скопируйте файлы проекта в директорию веб-сервера
2. Настройте права доступа к последовательным портам
3. Откройте веб-интерфейс в браузере

## Этапы калибровки

### 1. Настройки стенда
- Выбор COM-портов для всех устройств
- Настройка скорости передачи данных
- Проверка связи с оборудованием

### 2. Калибровка температуры
- Пошаговая калибровка по трем точкам: -40°C, +60°C, +10°C
- Автоматический контроль стабилизации температуры
- Запись эталонных значений в корректор

### 3. Поверка температуры
- Сравнение показаний корректора с эталоном (МИТ-8)
- Вычисление погрешности
- Автоматическая проверка допуска (±0.5°C)

### 4. Проверка импульсов
- Фиксация 20 импульсов
- Визуальный прогресс выполнения

### 5. Проверка давления
- Для абсолютного датчика: 5 точек (0%, 25%, 50%, 75%, 100%)
- Для датчика перепада: 3 точки (0%, 50%, 100%)
- Автоматический расчет погрешности
- Контроль попыток (максимум 3 на точку)

### 6. Финальный отчет
- Формирование протокола калибровки
- Экспорт в PDF
- Сохранение в базу данных
- Журнал действий с временными метками

## Визуальные принципы

- **Линейный мастер (Wizard)** - четкий индикатор прогресса
- **Цветовая индикация статусов:**
  - Серый - блок неактивен
  - Синий - блок в процессе
  - Зеленый - блок успешно завершен
  - Красный - ошибка, требуется вмешательство
- **"Большая кнопка оператора"** - для действий, требующих физического вмешательства

## Безопасность и контроль ошибок

- Таймауты на каждом этапе (15 минут бездействия)
- Полное логирование всех действий
- Временные метки для каждого события
- Возможность приостановки процесса

## Использование

1. Откройте `index.php` в браузере
2. Настройте параметры подключения к оборудованию
3. Нажмите "Проверить связь" для проверки устройств
4. Нажмите "Начать калибровку" и следуйте инструкциям мастера
